# DHL Supply Chain — Data Scientist Portfolio

Portfolio of data science projects rebuilt from real DHL supply chain experience. Demonstrates clustering, time-series forecasting, statistical testing, and optimization heuristics across six workstreams.

## Projects
1. SKU Segmentation — K-Means with ABC/XYZ overlay
2. Demand Forecasting — ARIMA with baseline comparison
3. RFM + A/B Test — segmentation and hypothesis testing
4. Warehouse Optimization — slotting, pick-path heuristic, market basket
5. WMS analytics — anomaly detection on operational metrics
6. Methodology — model cards, validation strategy, experiment design

**Status:** In progress.
